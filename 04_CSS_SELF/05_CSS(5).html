Flex레이아웃

웹페이지 레이아웃을 효율적으로 구성하기위한 기술.
flexbox를 사용하면요소들을 부모컨테이너에서 유연하게 정렬하고 배치할 수 있다. flex-box : 기본적으로 Flexbox는 한 줄에 자식 요소들을 강제로 배치합니다. 즉, 자식 요소들이 부모 요소의 너비를 넘어가더라도 줄 바꿈이 되지 않습니다. flex-wrap 속성을 사용하면, 부모 요소의 너비를 넘었을 때 자식 요소들이 줄 바꿈을 하며 새로운 줄에 배치될 수 있습니다.
