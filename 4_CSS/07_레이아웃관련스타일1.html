display : inline-block

기본적으로 inline 엘리먼트처럼 한 줄에 다른 엘리먼트들과 배치됨.
하지만 inline 엘리먼트에서 불가능하던 width와 height 속성 지정 및 margin과 padding 속성의 상하 간격지정이 가능함.
즉 inline의 성질을 가지면서 동시에 사용자가 원하는 대로 모형을 꾸밀 수 있다. button, input, select 태그 등이 대표적인 inine-block element이다.
