글꼴 관련 스타일 속성

font-family : 글꼴을 지정할 때 사용

선택자{font-family : 글꼴명;}

글꼴테스트 Font Style Test


font-size : 글꼴의 크기를 변경할 때 사용

선택자{font-size : 크기(px, em, rm, pt)}

font-weight : 글꼴의 굵기를 표현할 때 사용

선택자{font-weight: normal | bold | lighter | 100 ~ 900}
  • 원래 굵기
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 얇게

텍스트 관련 스타일(text~)

color : 텍스트 색상 지정

        선택자{
            color: 색상명 | #16진수(코드값) | rgb(x,x,x) | rgba(x,x,x,x) 
        }

text-decoration : 텍스트에 줄을 긋거나 줄을 없앨 때 사용

        선택자{
            text-decoration : none | underline | overline | line-through;
        }
  • 텍스트 영역 아래 줄 긋기
  • 텍스트 영역 중간에 줄 긋기
  • 텍스트 영역 위에 줄 긋기
네이버로

text-align : 텍스트 정렬할 때 사용

        선택자{
            text-align: left(기본값) | right | center | justify
        }
